# 模块一：连续型偏向随机初始化

传统的 SBI（Scaled Binomial Initialization） 方法主要用于二进制编码的个体初始化，其核心思想是通过控制 0 与 1 出现的概率分布，从而为不同个体注入不同的基因偏向，使得种群在初始化阶段即具备一定的“极端探索”能力。然而，在本文关注的施肥推荐系统中氮、磷、钾施肥量的直接建模中，二进制编码往往需要额外的编码与解码过程，这不仅增加了计算复杂度，而且在精度和边界控制上存在不足。为此，本研究将 Beta 分布引入初始化过程，提出基于 Beta 分布的连续型初始化方法。Beta 分布天然定义在 [0,1]区间，且其形状可通过参数 $\alpha$ 与 $\beta$ 灵活控制，因此能够直接生成具有“偏向性”的连续随机数。通过缩放映射到实际施肥量范围，可以避免二进制—连续映射的中间环节，更适合连续型多目标优化场景。具体初始化如图2 所示。

假设每个个体的染色体表示为：

$$
X = \left[ N _ {1}, P _ {1}, K _ {1}, \dots , N _ {s}, P _ {s}, K _ {s} \right]
$$

其中，S 表示施肥阶段数，每个变量 $x _ { j }$ 都有定义域：

$$
\min  _ {j} \leq x _ {j} \leq \max  _ {j}
$$

设种群规模为 $N$ ，第 $i$ 个体对应缩放因子为：

$$
s _ {i} = \frac {i - 0 . 5}{N}, i = 1, 2, \dots , N
$$

( 3 ) 

其中 $s _ { i } \to 0$ 表示该个体倾向于生成低值； $s _ { i } \to 1$ 表示该个体倾向于生成高值。这种线性缩放保证了种群从低施肥倾向到高施肥倾向的平滑覆盖。

为控制偏向程度，引入参数 $K$ （偏向强度系数），则：

$$
\alpha = 1 + K \cdot s _ {i}, \beta = 1 + K \cdot \left(1 - s _ {i}\right)
$$

( 4 ) 

当 $s _ { i } \approx 0$ 时， $\beta \gg \alpha$ ，Beta 分布在 0 附近集中，倾向于生成低施肥值；当 $s _ { i } \approx 1$ 时， $\alpha \gg \beta$ ，Beta分布在 1 附近集中，倾向于生成高施肥值；当 $s _ { i } \approx 0 . 5$ 时， $\alpha \approx \beta$ ，Beta 分布接近均匀分布，倾向于生成中间值。其中 $K$ 的大小控制了分布的集中程度： $K$ 较小时，分布更平滑、覆盖性更强； $K$ 较大时，分布更尖锐，个体更集中于边界解。

接下来对于个体 i，从Beta 分布中采样：

$$
r \sim B e t a (\alpha , \beta), r \in [ 0, 1 ]
$$

最后，将[0,1]区间的随机数映射至实际的施肥量范围：

$$
x _ {j} = \min  _ {j} + r \cdot \left(\max  _ {j} - \min  _ {j}\right)
$$

$$
(6)
$$

这样即可得到个体在某一阶段、某种养分的初始施肥量。由于 Beta 分布在 [0,1]区间具有极端解的聚集性，初始种群天然包含接近下界（极低施肥）和上界（极高施肥）的解，有助于探索帕累托前沿的两端。通过调节参数 $K$ ，可以灵活控制种群的分布特征，Beta 分布天然定义在 [0,1]，通过线性缩放后，生成的解始终在合法施肥范围内，避免了超界解的出现。

# 模块二：基于先验知识的结构化演化算子设计

为了使演化过程受农学机理正确引导，提出了基于精英样本原型引导的交叉算子和引入协同/拮抗先验的耦合变异算子两部分。其中，交叉算子利用历史高产施肥模式聚类得到的“施肥原型”作为引导锚点，使子代在继承父代多样性的同时向高质量施肥模式收敛，降低随机交叉带来的序列破坏；变异算子基于 N–P–K 之间的协同与拮抗关系构建耦合扰动，通过矩阵 M 显式编码养分间交互作用，使变异方向遵循农学机理。

# 步骤一、基于精英样本原型引导的交叉算子

在多目标进化算法NSGA-II的交叉算子中，常见做法是让两个父代个体在基因片段上进行信息交换。然而，这种方式在复杂时序施肥优化问题中存在随机性过强的问题，缺乏知识引导，父代之间的盲目组合可能破坏施肥序列中的时序协同关系，导致子代不可解释或无效。因此，本研究引入原型引导交叉，其核心思想是：在交叉过程中，不仅依赖父代个体的信息，还要引入高产施肥原型，使得子代生成更贴近历史数据中已验证的高产区域[13] 。模块结构图如图2 所示。

假设两个父代个体分别为 $X ^ { ( A ) }$ , $\boldsymbol { X } ^ { ( B ) } \in \boldsymbol { R } ^ { 3 T }$ ，其中 $T$ 表示施肥阶段数，每个阶段包含氮、磷、钾三个养分。设高产施肥原型集合为 A ={ A(1) , A(2) , … , A( M ) }，其中每个原型 $A { = } \{ A ^ { ( 1 ) } , A ^ { ( 2 ) } , { \ldots } , A ^ { ( M ) } \}$ ( k ) 来源于历史高产 $A ^ { ( k ) }$ 样本的聚类中心。

定义相似度（或距离）函数：

$$
d (P, A) = \sqrt {\sum_ {j = 1} ^ {3 T} \left(P _ {j} - A _ {j}\right) ^ {2}} \tag {11}
$$

即采用欧氏距离度量个体与原型之间的差异。

接着为父代寻找最匹配的原型，这里选择针对最优的父代选择离其最近的原型，并加入交叉策略中进行引导。首先在高产原型集合 A 中，寻找与当前父代中更优者 $X ^ { ( b e s t ) }$ 最接近的原型：

$$
k ^ {\star} = \underset {k} {\operatorname {a r g m i n}} d i s t \left(X ^ {(b e s t)}, A ^ {(k)}\right) \tag {12}
$$

其中， $d i s t ( \cdot , \cdot )$ 为 加权阶段距离，即：

$$
dist(X,A) = \sqrt{\sum_{s = 1}^{T}\omega_{s}\cdot\sum_{\substack{j\in N,P,K\\ (13)}}\left(x_{s}(j) - a_{s}^{(k)}(j)\right)^{2}}
$$

权重 $\omega _ { s }$ 反映各阶段对产量或目标的重要性。

对每个阶段 s 与养分 $j$ ，先进行父代的凸组合：

$$
x _ {s} ^ {(C)} (j) = \alpha_ {s} x _ {s} ^ {(A)} (j) + (1 - \alpha_ {s}) x _ {s} ^ {(B)} (j) \tag {14}
$$

其中 $\alpha _ { s } \sim B e t a ( 2 , 2 )$ ，保证凸组合更集中于中间区域；

在父代融合结果的基础上，引入原型 A ( k  ⋆  ) $A ^ { ( k ^ { \ast } ) }$ 的牵引：

$$
x _ {s} ^ {(C)} (j) \leftarrow \left(1 - \gamma_ {s}\right) x _ {s} ^ {(C)} (j) + \gamma_ {s} a _ {s} ^ {(k ^ {\cdot})} (j) \tag {15}
$$

其中牵引系数 $\gamma _ { s }$ 定义为：

$$
\gamma_ {s} = \gamma_ {0} \cdot u _ {g} \cdot r _ {s} \tag {16}
$$

其中 $\gamma _ { 0 }$ 为初始牵引强度； $u _ { g } \in \left( 0 , 1 \right.$ 为退火因子，随代数逐步衰减，使模型早期更依赖原型，后期更自由探索。

为了保证子代在约束范围内，最终对子代执行可行域投影，确保所有解满足施肥上下界限制。

$$
x _ {s} ^ {(C)} (j) \leftarrow \min  \left(m a x \left(x _ {s} ^ {(C)} (j), m i n _ {j}\right), m a x _ {j}\right) \tag {17}
$$

上述交叉算子的设计保证解朝向历史高产模式靠拢，减少无效探索，最终生成的子代既继承了父代的变异性，又继承了原型的优良模式，实现知识引导和自适应搜索的平衡。

# 步骤二、协同/拮抗先验的耦合变异

在作物施肥优化中，氮（N）、磷（P）、钾（K）三大养分之间存在复杂的交互作用，如氮和磷之间常表现为正向互补的协同效应，联合增加可显著提高作物吸收效率，而过量的铵态氮肥( N H+¿)¿ 会与钾离子 ( $( N H _ { 4 } ^ { + i , ) i }$ K+¿)¿ 形成一定的竞争关系，表现为拮抗效应。传统的独立突变算子往往假设 $\left( K ^ { + i \setminus j \bar { \zeta } } \right.$ 

养分扰动互相独立，无法捕捉这些 “协同/拮抗先验”，容易产生不符合农学规律的方案。为此，提

出协同/拮抗先验的耦合变异，通过显式引入交互矩阵和状态自适应机制，把农业知识转化为统计结构，实现更合理的联合变异。详细如图3所示。

首先定义协同拮抗交互矩阵M：

$$
M = \left[ \begin{array}{c c c} 0 & \rho_ {N P} & \rho_ {N K} \\ \rho_ {P N} & 0 & \rho_ {P K} \\ \rho_ {K N} & \rho_ {K P} & 0 \end{array} \right] \tag {18}
$$

其中 $\rho _ { i j }$ 的正负表征了作用的类型，值为正则为协同作用，在进行变异的时候将被引导为同向变化，相对的为拮抗作用，不宜同时作大幅度同向波动。 $M$ 矩阵的制定由经验指定，辅以历史数据统计。变异算子选择多元高斯扰动，根据耦合结构，在每个作物时期 $t$ ，对施肥向量 $\scriptstyle { z _ { t } = \left( N _ { t } , P _ { t } , K _ { t } \right) }$ 进行多元高斯扰动，

$$
\Delta z _ {t} \sim N \left(0, \sigma^ {2} I + \alpha D _ {t} M D _ {t}\right) \tag {19}
$$

其中， $\sigma ^ { 2 } I$ 为基线独立扰动， $M$ 为上文定义的耦合结构， $\alpha$ 为知识引导强度， $D _ { t }$ 为缩放系数矩阵。相比传统的独立高斯扰动（仅 $\dot { \cdot } _ { N } ( 0 , \sigma ^ { 2 } I )$ N 0,σ2I ），该设计可在多维耦合变量的优化中更有效地保持搜索分布与解空间流形的一致性，提升策略更新的收敛速度与稳定性。

由此得到变异后的施肥向量：

$$
\mathrm {z} _ {t} ^ {\prime} = \mathrm {z} _ {t} + \Delta \mathrm {z} _ {t} \tag {20}
$$

在突变触发层面，定义单元变异概率为：

$$
p _ {t} = p _ {0} \left[ 1 + \beta \cdot \stackrel {\text {d e f}} {=} (t) + \gamma \cdot \operatorname {S e n s} (t) + \delta \cdot \operatorname {D i v P r e s s} \right] \tag {21}
$$

其中 $\dot { \boldsymbol { \mathscr { L } } } \left( t \right)$ 为缺素/比例偏离分数，土壤诊断或父本 $N { : } P { : } K$ 失衡时增大；Sens(t)为物候阶段敏感度（如灌浆期对 $N$ 敏感，则提升触发概率，但步长适度缩小）；DivPress种群多样性压力（若收

敛加快，则增大扰动概率）。这样可实现当养分更缺、阶段更敏感、更单一时更可能突变的自适应特征。

通过矩阵 M 的正负相关性，把“协同-拮抗规律”融入多元正态扰动的协方差结构，将农业知识内化为统计结构，同时兼顾缺素程度、作物阶段敏感度等动态因素，从而调控变异的程度，达到既保证种群多样性的同时，又保证样本尽可能的拟合现实农学先验知识，避免盲目扰动。

# 模块三：动态精英保留策略

首先依据产量最大化、环境影响最小化与施肥成本最小化三个目标函数，完成多目标适应度评估，并采用非支配排序获得个体的帕累托层级。在此基础上，本模块构建了结合目标空间与决策空间的综合距离度量，用于刻画个体的多样性结构并计算局部稀疏度。随后，引入动态多样性权重机制，构建随迭代过程逐步减弱的探索–收敛调节因子，并结合种植阶段的差异化需求构成阶段性调节函数。最终将非支配等级与动态稀疏度因子整合为精英得分，实现对精英个体的动态筛选与保留，从而在收敛性与施肥模式多样性之间维持平衡。

# 步骤一：多目标与适应度计算

在实际生产过程中，农业决策者需要依据当前状况选择最优的施肥策略，需要关注多个目标问题，本算法主要考虑最大化产量、最小化成本以及最小的环境影响这三个目标。多目标函数定义如下：

$$
F (x) = \left[ \begin{array}{c} f _ {1} (x), f _ {2} (x), f _ {3} (x) \end{array} \right] ^ {\top} \tag {7}
$$

其中， $f _ { 1 } ( x )$ 指代产量目标，在这里可以将其转换为最小化形式，即 $\stackrel { \sim } { f } _ { 1 } ( x ) = - Y \left( x \right) \stackrel { \bullet } { ^ { \prime } } f _ { 2 } \left( x \right)$ 指代环境影响目标，在农业实践中，主要考虑氮肥的施放对环境的影响，即氮素问题； $f _ { 3 } ( x )$ 指代成本目标。下面逐一给出适应度的实现方法。

产量 目前有许多作物模型可用作优化目标函数，而 DSSAT 无疑是最佳选择，因为它易于使用且经过充分验证。用户通过输入定义的土壤、气象、品种和作物管理文件来运行模型，该模型模拟在管理条件下，同一地块上作物的生长、发育和产量，以及在不同耕作制度下土壤水分、碳和氮随时间的变化。但单次 DSSAT 模拟较慢，不能在进化循环中频繁调用。本文采用代理模型 ${ \widehat { Y } } \left( x \right)$ 来近似DSSAT 输出，并在算法运行中主要使用 Y^，同时采用基于不确定性的泛化策略，以预算内的高置信度点替换代理，控制代理偏差[11] 。

代理模型选择 LightGBM （Light Gradient Boosting Machine）模型[12] ，是由微软提出的一种基于梯度提升树的高效实现，采用基于直方图的分裂方法，大幅降低了训练时间与内存占用，同时能有效应对非线性和多维特征交互关系。具体近似如下：

$$
\widehat {Y} (x; \theta) \approx Y ^ {D S S A T} (x) \tag {8}
$$

训练集由 n 个 DSSAT 真值对 $D = \{ \left( \boldsymbol { x } ^ { ( i ) } , \boldsymbol { Y } ^ { ( i ) } \right) \} _ { i = 1 } ^ { n }$ 构成。

训练目标为最小化均方误差：

$$
\theta^ {i} = \arg \min  _ {\theta} \frac {1}{n} \sum_ {i = 1} ^ {n} \left(\hat {Y} \left(x ^ {(i)}; \theta\right)\right) - Y ^ {(i)}) ^ {2} + \lambda R (\theta)
$$

其中 $R$ 为正则化项，?? 为超参数。

环境影响 在这里本文主要考虑的是氮素流失问题，该目标也可以通过 DSSAT算法直接输出，故同产量目标函数一般处理，首先通过过程模型获得真值对，然后训练代理模型。

成本 成本由可分解的两部分组成：变量成本（随施肥量变化）与固定成本（与工序/设备/人工相关，常视为常数），成本函数由下式定义：

# 步骤二、基于综合距离的动态稀疏度度量

传统NSGA-II算法的一个重要部分就是非支配排序，并辅以拥挤距离的定义，最终帮助算法确定帕累托前沿。本模型在其基础上，引入了额外的机制，使得精英保留策略

首先在距离方面，为了保持解在目标空间的均匀分布，需要最大化样本点之间的多目标空间距离；同时考虑不同施肥策略时序施肥距离，约束施肥曲线模式的多样性。对于两个个体 i， $j$ ，定义综合距离：

$$

$$

$$
f _ {i} = \left[ f _ {1} \left(x _ {i}\right), f _ {2} \left(x _ {i}\right), f _ {3} \left(x _ {i}\right) \right]
$$

$$
\omega_ {f} + \omega_ {x} = 1
$$

其中 $f _ { i }$ 代表个体 $i$ 多目标向量， $x _ { i }$ 代表该个体的NPK施肥策略， $f _ { p } ( \cdot )$ 代表研究的多目标任务函数，第二部分的 $x _ { i } ( t ) - x _ { j } ( t )$ 则是为了量化施肥模式，约束施肥曲线模式的多样性， $\omega _ { f }$ 和 $\omega _ { x }$ 为两者之间的平衡权重。
